# Kabeer Mahmood — Portfolio

A personal portfolio website built with pure HTML and CSS. This is a Week 1 project focused on practicing semantic HTML, CSS Flexbox, CSS Grid, and responsive design.

## Live Sections

- **Header & Navigation** — Name, subtitle, and smooth-scrolling nav links (Flexbox)
- **About** — Short bio with a profile avatar
- **Skills** — Grid of 6 technology cards (HTML, CSS, JavaScript, React, Python, PostgreSQL)
- **Projects** — Grid of project cards with descriptions and links
- **Contact** — Clean form with Name, Email, and Message fields
- **Footer** — Copyright notice

## Responsive Breakpoints

| Screen        | Width    | Grid Columns |
|---------------|----------|--------------|
| Desktop       | 769px+   | 3 per row    |
| Tablet        | 481–768px| 2 per row    |
| Mobile        | ≤480px   | 1 per row    |

Navigation links stack vertically on mobile.

## Tech Stack

- HTML5 (semantic elements)
- CSS3 (Flexbox, Grid, media queries)
- No JavaScript, frameworks, or libraries
